= SMO profile show 명령
:icons: font
:imagesdir: ../media/


[role="lead"]
profile show 명령을 실행하여 프로파일에 대한 정보를 표시할 수 있습니다.



== 구문

[listing]
----

        smo profile show
-profile profile_name
\[-quiet \| -verbose\]
----


== 매개 변수

* * - profile profile_name *
+
프로파일의 이름을 지정합니다. 이 이름은 최대 30자까지 지정할 수 있으며 호스트 내에서 고유해야 합니다.

* * -저소음 *
+
콘솔에 오류 메시지만 표시합니다. 기본값은 오류 및 경고 메시지를 표시하는 것입니다.

* * - 자세한 정보 표시 *
+
콘솔에 오류, 경고 및 정보 메시지를 표시합니다.
