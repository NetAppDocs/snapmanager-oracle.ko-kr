= Oracle용 SnapManager 설치
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용자 환경에서 SnapManager for Oracle을 다운로드하여 설치하고 데이터베이스 백업, 복원, 복구, 클론 복제 등의 작업을 수행할 수 있습니다.

SnapManager for Oracle 설치 패키지에는 호스트 서버 소프트웨어와 그래픽 사용자 인터페이스(GUI) 클라이언트 소프트웨어가 포함됩니다.
