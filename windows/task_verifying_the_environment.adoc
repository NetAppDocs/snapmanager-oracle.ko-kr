= 환경을 확인하는 중입니다
:icons: font
:imagesdir: ../media/


[role="lead"]
환경을 확인하여 SnapDrive 및 SnapManager가 올바르게 설정되었는지 확인할 수 있습니다.

필요한 필수 구성 요소를 다운로드, 설치 및 설정합니다. SnapManager가 설치되어 있고 호스트 서버가 실행 중인지 확인합니다.

. SnapDrive가 설치되어 있고 루트 계정에서 실행할 수 있는지 확인하려면 SMO system verify 명령을 실행합니다
