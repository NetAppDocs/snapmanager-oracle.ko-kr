= 프로파일을 삭제하는 중입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
성공했거나 불완전한 백업이 포함되어 있지 않으면 언제든지 프로필을 삭제할 수 있습니다. 확보되거나 삭제된 백업이 포함된 프로필을 삭제할 수 있습니다.

. 프로필을 삭제하려면 SMO profile delete -profile profile profile_name 명령을 입력하십시오
