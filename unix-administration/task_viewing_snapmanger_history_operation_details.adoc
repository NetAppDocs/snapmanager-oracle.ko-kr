= SnapManager 기록 구성 세부 정보 보기
:icons: font
:imagesdir: ../media/


[role="lead"]
단일 프로파일의 기록 설정을 볼 수 있습니다.

SnapManager 기록 작업은 각 프로필에 대해 다음 정보를 표시합니다.

* 작업 이름입니다
* 보존 클래스
* 보존 수입니다
+
.. 특정 profile에 대한 SnapManager history 동작 정보를 표시하려면 'smo history show -profileprofile_name' 명령어를 입력한다
