= SMO 프로필 목록 명령
:icons: font
:imagesdir: ../media/


[role="lead"]
이 명령은 현재 프로파일 목록을 표시합니다.



== 구문

[listing]
----

        smo profile list
\[-quiet \| -verbose\]
----


== 매개 변수

* * -저소음 *
+
콘솔에 오류 메시지만 표시합니다. 기본값은 오류 및 경고 메시지를 표시하는 것입니다.

* * - 자세한 정보 표시 *
+
콘솔에 오류, 경고 및 정보 메시지를 표시합니다.
